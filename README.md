📦 StockWise – Inventory Management System

StockWise is a secure and modular inventory management system built with Spring Boot, designed to manage products, categories, suppliers, and stock operations with real-world business logic.

🚀 Features
🔐 JWT-based Authentication & Authorization
👥 Role-based access control (ADMIN, USER)
📦 Product, Category, Supplier management
🛒 Purchase system (stock increase logic)
📊 Minimum stock level tracking
🔍 Pagination & search support
⚠️ Global exception handling
✅ Request validation (Bean Validation)
📄 Swagger/OpenAPI documentation
🐳 Dockerized application (Docker Compose)
